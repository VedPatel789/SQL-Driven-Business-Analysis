# SQL-Driven Business Analysis Dashboard

##  Overview
This project is a browser-based SQL analytics dashboard that demonstrates business intelligence using structured queries, data visualization, and interactive reporting.

It simulates real-world business analysis by processing customer, order, and revenue data.

---

## Key Features
- SQL query execution in browser (SQLite)
- Interactive query editor
- Real-time charts (Chart.js)
- Customer & revenue analytics
- Business insights dashboard

---

## Project Structure
- data/ → database schema & sample data
- sql/ → SQL queries for analysis
- src/ → application logic
- assets/ → UI branding
- screenshots/ → output previews
- docs/ → documentation & reports

---

## Business Insights Covered
- Total revenue analysis
- Top customers ranking
- Sales distribution
- Customer segmentation
- Order performance tracking

---

## Tech Stack
- HTML, CSS, JavaScript
- SQL.js (SQLite in browser)
- Chart.js (visualization)
- GitHub Pages (deployment)

---

## How to Run
1. Clone repository
2. Open `index.html` in browser
3. Run SQL queries using editor
4. View charts and results
